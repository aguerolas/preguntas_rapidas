# executive

1.	¿Cuál es la IP del servidor destino al que se conectará WinSCP?
	2.	¿Cuál es la IP de la workstation desde donde se hará la transferencia?
	3.	¿Cuál es el usuario o usuarios SFTP que se usarán para conectarse?
	4.	¿Cuál es el nombre completo del usuario designado para hacer la transferencia?
	5.	¿Cuál es el nombre y la IP interna del servidor donde se consolidan los archivos antes de transferirlos?
	6.	¿Qué tipo de archivos se van a transferir y cuál es su contenido o propósito?
	7.	¿A qué entidad pertenece el servidor destino (regulador, proveedor, entidad interna, etc.)?
	8.	¿Cuál es la razón de urgencia del request? ¿Es un requerimiento regulatorio, operacional, contractual?
	9.	¿Qué consecuencias habría si no se completa la transferencia en el plazo requerido?
	10.	¿Qué sistema genera los archivos a transferir (AS-400, otro)?
	11.	¿Cuál es el área o departamento responsable de ejecutar la transferencia?
	12.	¿Cuál es el nombre del sistema de información asociado a este proceso?

•	ISO 27001 – A.13.2 (Information Transfer): Requires formal policies and controls for data transfer.
	•	ISO 27001 – A.12.4 (Logging and Monitoring): Requires event logging and monitoring of system activities.
	•	NIST SP 800-53 – SC-8 (Transmission Confidentiality and Integrity): Mandates protection of transmitted information.
	•	NIST SP 800-53 – AU-2 / AU-12 (Audit Events / Audit Record Generation): Requires auditable logging of file transfer activity.
	•	PCI-DSS – Requirement 10 (Log and Monitor All Access): All access to network resources must be logged and monitored.
	•	Internal Bank Policy – Standardized Tooling Ecosystem: Mandates the use of Control-M as the approved file transfer mechanism.

# executive

Actualmente, la entidad financiera Crediscotia fue vendida a Santander Consumer, otra entidad bancaria, tras dejar de formar parte del grupo Scotiabank. Como parte de esta transición, los requisitos y proyectos de la entidad se centran exclusivamente en cuestiones regulatorias y de ciberseguridad para poder asegurar el cumplimiento normativo y la adecuada gestión de sus activos tecnológicos bajo el nuevo marco corporativo de Santander.

De esta manera, la solución propuesta por SBP se está adaptando a la infraestructura y arquitectura de Crediscotia, con el fin de alinear sus capacidades tecnológicas a los estándares del grupo. Como parte de este proceso, se ha identificado que Crediscotia cuenta con un conjunto de aplicaciones legacy que requieren evaluaciones de seguridad.

# executive

Cumplir con la actualización del parámetro de fecha de corte en la base de datos de la aplicación CrediScotia Financiera Mobile App. Se realizará la implementación del script sobre la tabla HBCSF.PARAMETER para actualizar el campo starting_date del grupo PUSH_NOTIFICATION, así como el registro de la nueva versión 3.3.1 en la tabla HBCSF.VERSION_DETAILS. El impacto de no implementarse el cambio podría generar inconsistencias en el comportamiento de las notificaciones push de la aplicación móvil. Es importante remarcar que no se realizan cambios a nivel de código de la aplicación ni de servicios, limitándose únicamente a una actualización de datos en base de datos. El escaneo de seguridad adjunto no identifica vulnerabilidades. Por lo tanto, se concluye que este cambio no requiere la elaboración de un nuevo TRA.

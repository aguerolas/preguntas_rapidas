Risk Description (paraphrased):

A potential data leak or loss at Santander is suspected due to the absence of DLP security controls over the mailbox “crediscotiacorreo@crediscotia.com.pe”. This means that the bank’s security tools do not validate or review the information before it is sent, resulting in a lack of control over the data being transmitted.


Action Plan (translated):

As part of the TSA support and guidelines, this application will be decommissioned upon its expiration (including the shutdown of application services and server seizure if necessary), that is, by April 2027; as a consequence, the mailbox will also be decommissioned.

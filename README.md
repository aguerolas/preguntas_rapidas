# executive

El plan de remediación implica transitar hacia una solución automatizada de transferencia de archivos con gestión centralizada de credenciales, lo cual requiere el inicio de una nueva iniciativa formal. Los usuarios de negocio actuales acceden al proceso bajo los controles compensatorios vigentes, y la regularización del mecanismo de transmisión debe seguir el flujo estándar de gestión de proyectos de la organización.


El IT Owner ha confirmado su compromiso con la remediación; sin embargo, la fecha de marzo 2027 responde a las restricciones actuales de capacidad del equipo. En este momento se están gestionando múltiples iniciativas de cumplimiento regulatorio de alta prioridad, y la implementación de este plan requiere pasar por el flujo regular de gestión de proyectos, lo cual determina dicho plazo.
